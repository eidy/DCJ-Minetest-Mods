Currency by Devyn Collier Johnson (DevynCJohnson@Gmail.com)


Description:
This mod provides coins that users may use to establish an economy.


Recipes (shapeless):

1mineunit (currency:1mineunit) = default:copper_ingot + default:copper_ingot
10mineunit (currency:10mineunit) = default:bronze_ingot + default:bronze_ingot
100mineunit (currency:100mineunit) = default:gold_ingot + default:gold_ingot
1000mineunit (currency:1000mineunit) = default:mese_crystal + default:mese_crystal

Cooking reverses the crafting


Mod dependencies: default
